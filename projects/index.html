Projects

#NEWPALMYRA

"Help Build a future #NEWPALMYRA from the past." A project co-founded by Bassel before he was detained. Join the project and help pick up where Bassel left off.

http://newpalmyra.org

Cost of Freedom Book

Collaborative book written in honor of Bassel about the real costs associated with Freedom.

http://costoffreedom.cc

Waiting Poetry Book

Book written by Noura Ghazi Safadi, Bassel's wife, and translated by Bassel into English, about the experience of waiting on a loved one.

http://waiting4bassel.cc

Bassel Tweets

A visual display of Bassel's tweets over time.

http://kiki.wiki/bassel/

Freebassel API

Originally Bassel Days project showing the number of days Bassel has been detained, but now expanded to display that and now the number of days he has been disappeared.

http://api.freebassel.org

Freebassel Timeline

A timeline of Bassel's life since birth until present day. If you have more knowledge about Bassel, please add to the growing timeline.

http://timeline.freebassel.org
